String Dtypes - refer to pdf

	a. ENUM: only predefined values van be inserted

	ex:
	ALTER TABLE dt_demo ADD COLUMN gender ENUM('male' , 'female' , 'others);

	UPDATE dt_demo
	SET gender = 'delhi'

	THIS VALUE WONT BE UPDATED INSTEAD EMPTY STRING WILL REPLACED;

-----------------------------------------------------------------------

	b. SET : SIMILAR TO ENUM , BUT CAN STORE MULTIPLE VALUES
	ex: hobbies can be multiple [cricket , dancing , singing]

	QUERY:
	INSERT INTO dt_demo (hobby) VALUES
	('sports') , ('gaming') , ('sports , gaming');

------------------------------------------------------------------------

	C. BLOB: refer to pdf

	QUERY: ALTER TABLE dt_demo ADD COLUMN dp MEDIUMBLOB;
	INSERT iNTO dt_demo (dp) VALUES (LOAD_FILE(path))

NOTE : PATH SHOULD CONTAIN FORWARD SLASHES
